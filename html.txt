TML とは何か、プログラミング初心者にわかるように説明してください。
TML（HyperText Markup Language）は、ウェブページを構築するための基本的な言語です。これは、テキストや画像、リンクなどのコンテンツを構造化し、ブラウザに表示するためのマークアップ言語です。例えば、<p>タグは段落を表し、<img>タグは画像を表示します。HTMLを使用することで、ウェブページの骨組みを作成し、他のプログラミング言語やスタイルシートと組み合わせることができます。

CSS とは何か、プログラミング初心者にわかるように説明してください。
CSS（Cascading Style Sheets）は、ウェブページの見た目やスタイルを定義するための言語です。HTMLがページの構造を担当するのに対し、CSSはそのデザインや装飾を担当します。例えば、文字の色やフォント、背景の色、要素の配置などを指定することができます。CSSはHTMLと組み合わせて使用され、ウェブページを美しく整え、使いやすくするのに役立ちます。

HTML と　CSS はどのような関係にあるのか、プログラミング初心者にわかるように説明してください。
HTMLとCSSはウェブページを作成する際に協力しています。HTMLはページの構造を定義し、要素やコンテンツを配置します。一方、CSSはそのHTML要素にスタイルやデザインを与え、見た目を整えます。
具体的に言うと、HTMLはページの「骨組み」であり、段落、見出し、画像などの基本的な構造を指定します。CSSはこれらの要素に対して、色、フォント、余白、配置などのスタイルを指定し、ページを美しく装飾します。
このように、HTMLとCSSは連携してウェブページを作り上げ、HTMLが構造を提供し、CSSがそのデザインを担当する役割分担があります。
